# 🎮 Hangman Game (C++)

A console-based **Hangman Game** developed in **C++** using **Visual Studio** as part of the **Object-Oriented Programming** course.

The project implements the classic Hangman game, where players guess letters or the entire word while a hangman figure is gradually drawn after incorrect guesses.

---

## 📖 Overview

The game randomly selects a word from a predefined word list. Players try to reveal the hidden word by guessing letters. After each incorrect guess, another part of the hangman is drawn. Players may also choose to guess the complete word before using all available attempts.

The project demonstrates the use of functions, arrays, strings, loops, conditional statements, and random number generation in C++.

---

## ✨ Features

- Random word selection
- Letter-by-letter guessing
- Full word guessing option
- Hangman drawing after incorrect guesses
- Display of guessed letters
- Display of the current word status
- Win and lose scenarios
- Restart game option
- Console-based interface
